# Getting Started

## Step 1: Setup Project

To setup the project clone the repository and then run following command:

```bash
# Using Yarn
yarn install
```

## Step 2: Start Application

Run the following command to start Android or iOS app:

### For Android

```bash
# Using Yarn
yarn android
```

### For iOS

```bash
# Using Yarn
yarn ios
```
